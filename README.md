Project contains:
1. engine.py: A fully implemented chess engine that searches for all legal moves in a given position and encodes each move with a unique index of 0-4672. The engine's performance reaches 1024 games of 200 half-moves in approximately 48 seconds.
2. models.py: The model is a compact convolutional neural network consisting of an input convolutional layer (12→64 channels), four residual blocks, and two independent output heads. The policy head predicts the logits for all 4672 possible actions, and the value head calculates the probability of the current player winning in the range from -1 to 1 using the tanh function. The Adam optimizer is used for training.
3. train.py: This code implements self-play training for a chess agent using the Actor-Critic architecture. Several parallel processes independently generate games, in which the neural network selects moves using a stochastic policy with temperature sampling, which ensures a balance between exploration and the use of accumulated experience. After completing the games, target values ​​for the value function and advantage are calculated, which are used to update both heads of the network: policy minimizes the policy gradient loss, and value is trained using the Huber loss function.
